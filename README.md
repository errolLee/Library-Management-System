# 📚 Library Management System

## 📌 Overview
This is a Windows Forms application developed in C# for managing a library system. It allows users to manage books, borrowers, and transactions efficiently.

---

## ✨ Features
- User Login System (Admin / User)
- Add, Edit, Delete Books
- Manage Borrowers
- Search Books (real-time filtering)
- Borrow and Return System
- Data stored using MySQL Database

---

## 🛠️ Technologies Used
- C# Windows Forms
- MySQL Database
- Visual Studio

---

## 🚀 How to Run
1. Open the solution file (.sln) in Visual Studio
2. Restore NuGet packages if needed
3. Configure database connection in code
4. Run the project
